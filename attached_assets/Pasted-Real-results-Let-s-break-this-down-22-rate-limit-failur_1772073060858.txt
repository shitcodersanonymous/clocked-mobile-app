Fix 2: Exact combo preservation (B1.2, B8)

The auto-repair in aiAutoRepair.ts clamps max duration to 300s. That's correct for boxing rounds but wrong for continuous activities like treadmill runs. Fix the clamping logic:
typescript// Instead of blanket 300s cap:
if (segment.segmentType === 'exercise' && isContinuousActivity(segment.name)) {
  // Allow up to 3600s (60 min) for treadmill, jump rope, rowing, etc.
  segment.duration = Math.min(Math.max(segment.duration, 10), 3600);
} else {
  segment.duration = Math.min(Math.max(segment.duration, 10), 300);

typescriptlet attempts = 0;
